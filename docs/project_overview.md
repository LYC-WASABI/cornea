# Project Overview

## Purpose

This repository documents the cornea-lid mixed lubrication workflow around:

- Thin-Film Flow with JFO/Elrod-Adams cavitation
- Solid Mechanics contact between lid and cornea
- load sharing between film pressure and contact force
- staged continuation from static baseline to dynamic recursive coupling

## Main repository layers

- root `*.md`
  - stage notes and milestone summaries
- root `*.java`
  - COMSOL builder, probe, diagnose, and verification scripts
- `CONTEXT.md`
  - curated reading order for external ChatGPT/Codex sessions
- `LATEST_MODEL.md`
  - current latest-attempt versus latest-validated distinction
- `docs/*.md`
  - stable project-context files for ongoing collaboration

## Current baseline understanding

- Stage 200 is the trusted static or quasi-static reference model family.
- Stage 576 is the current dynamic recursive coupling development line.
- The latest experimental branch is not automatically the best or validated one.
