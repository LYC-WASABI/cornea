# 385_lid8mm_stage200_official_jfo_joint_load_results_Model 说明文档

## 1. 模型定位

本模型文件：

```text
385_lid8mm_stage200_official_jfo_joint_load_results_Model.mph
```

是当前较干净的“角膜-刮擦头-泪膜混合润滑”静态/准静态载荷平衡版本。模型目标不是最终完整动态刮擦，而是建立一个可继续发展的基准状态：

```text
液膜承载力 + 固体接触力 ≈ 0.03 N
```

该版本已经将薄膜压力从简单的 `max(pfilm,0)` 修正为 COMSOL 官方 JFO/Elrod-Adams 空化模型下的物理压力 `tff.p`，并在静态参考位置实现了较好的法向载荷分担。

## 2. 当前已完成的模型内容

### 2.1 固体结构

模型中已包含：

- 3D 实体角膜球冠。
- 3D 柔性弧形 lid wiper。
- 角膜缘固定约束。
- 角膜后表面眼压。
- 角膜后表面弹性基础/等效支撑。
- 显式接触对：
  - Source：lid wiper 内侧接触面。
  - Destination：角膜前表面。
- Solid Mechanics 接触计算。
- 可输出角膜和 lid wiper 的位移、von Mises 应力、接触压力和接触力。

### 2.2 薄膜流

模型中已加入：

- `Thin-Film Flow, Shell`
- 基准泪膜厚度：

```text
h0_tear = 3 um
```

- 残余数值膜厚：

```text
h_residual = 0.1 um
```

- JFO/Elrod-Adams 空化：

```text
Equation type = Reynolds Equation with Cavitation
```

后处理中应使用：

```text
tff.p       物理液膜压力
tff.theta   液相填充率
```

不要直接将内部求解变量 `pfilm` 当作物理压力。

### 2.3 载荷分担

Stage 200 的关键结果为：

```text
平均有效膜厚 havg = 3.100213 um
液膜承载力 Wfilm = 0.0284908 N
固体接触力 Fn_contact = 0.00150055 N
总法向力 Ftotal = 0.0299913 N
液膜剪切力 FshearFilm = 1.91555e-4 N
液膜摩擦系数 muFilm = 0.00639
平均 lambda = 6.195
平均液相率 theta = 0.99985
```

说明静态参考位置下，模型已经能够实现：

```text
Wfilm + Fn_contact ≈ 0.03 N
```

## 3. 主要参数含义

| 参数/变量 | 含义 | 当前作用 |
|---|---|---|
| `F_total_target` | 目标总法向载荷 | 目标为 `0.03 N` |
| `h0_tear` | 基准泪膜厚度 | 当前为 `3 um` |
| `h_residual` | 数值残余膜厚 | 防止膜厚为零导致奇异 |
| `delta_h_jfo197` | 液膜法向分离/等效膜厚调整量 | Stage 200 中用于校准 JFO 液膜承载 |
| `p_cav_transition195` | JFO 空化过渡平滑宽度 | 用于稳定 Elrod-Adams 空化求解 |
| `beta_tear195` | 空化模型中的等效可压缩性/正则化参数 | 帮助 JFO 方程在空化区保持数值可解 |
| `tff.p` | JFO 模型下的物理液膜压力 | 用于计算液膜承载 |
| `tff.theta` | 液相填充率 | 判断空化或非满膜区域 |
| `Fn_contact119` | 固体接触压力积分得到的法向接触力 | 表示角膜和刮擦头粗糙峰/实体接触承载 |
| `Wfilm199` | 液膜压力积分得到的法向承载力 | `intop_film(tff.p)` |
| `Ftotal199` | 总法向承载力 | `Fn_contact119 + Wfilm199` |
| `dr_indent119` | 可调压入量 | 通过全局方程调节刮擦头压入 |
| `FshearFilm199` | 液膜剪切力积分 | 用于计算液膜摩擦贡献 |
| `lambda` | 膜厚与等效粗糙度比值 | 判断润滑状态 |
| `theta_lid` / `phi_lid_structure` | 刮擦头角位置 | 用于控制刮擦头沿角膜表面运动 |
| `omega_lid` | 刮擦头角速度 | 用于薄膜上壁速度 |

## 4. 每一步建模的意义

### 4.1 建立角膜和刮擦头固体模型

意义：

- 提供真实的软组织接触变形。
- 允许输出角膜位移、应力和接触压力。
- 为后续摩擦力和载荷分担提供结构基础。

如果没有固体模型，只做薄膜流，无法得到角膜应力和接触力。

### 4.2 建立显式 Contact Pair

意义：

- 明确接触源面和目标面。
- 避免 Form Union 导致的面合并问题。
- 使接触压力、gap distance 和接触力可以被单独后处理。

当前接触对的物理含义是：

```text
lid wiper 内侧面压向角膜前表面
```

### 4.3 使用可调压入量

意义：

刮擦头不是简单施加固定压力，而是通过压入量改变结构接触和液膜压力，使总载荷满足：

```text
Fn_contact + Wfilm = 0.03 N
```

这比直接给角膜表面施加等效压力更接近实验中的“刮擦头压在眼表上”的情况。

### 4.4 加入 Thin-Film Flow, Shell

意义：

- 将泪膜作为微米级薄膜处理。
- 避免建立真实 3D 微米流体体积导致网格极端困难。
- 可以计算液膜压力、剪切力和局部空化。

### 4.5 使用 JFO/Elrod-Adams 空化

意义：

普通 Reynolds 方程可能得到不合理负压力。JFO 空化通过液相率 `theta` 处理非满膜区域，可以保持质量守恒。

当前重要变量：

```text
tff.p       物理压力
tff.theta   液相填充率
```

### 4.6 液膜压力反馈到结构

意义：

液膜压力不只是后处理结果，而是作为法向载荷反馈给结构：

```text
液膜压力载荷 + 固体接触 = 总压入力
```

这样才能表达“刮擦头的 0.03 N 由角膜实体接触和泪膜共同承担”。

### 4.7 分区求解策略

当前采用：

```text
先求液膜压力
再冻结 tff.p
再求结构接触和压入量
```

意义：

- 避免将接触、JFO 空化、全局压入控制放进同一个强耦合牛顿矩阵。
- 减少 `pfilm` 空行、接触分支跳变和一致初值失败。
- 更适合逐角度准静态扫描。

## 5. 当前模型的不足

### 5.1 薄膜作用区域仍然过大

当前 Thin-Film Flow 仍主要作用于整个角膜前表面，而不是局部刮擦轨迹带。

问题：

- 液膜压力可以在整个角膜范围内建立。
- 排液边界位于远端角膜缘。
- 刮擦区附近没有真实入口、出口和侧向排液边界。
- 动态刮擦时容易形成不真实的长程液动轴承。

后续 Stage 205–209 诊断已经显示：

```text
刮擦头移动到 -34.5 deg 后，液膜承载可达到 0.2-0.6 N
```

远高于目标：

```text
0.03 N
```

### 5.2 当前静态平衡不能代表完整动态刮擦

Stage 200 只是静态/准静态参考点。

它证明了：

```text
某一参考位置可以实现液膜 + 接触共同承担 0.03 N
```

但还没有证明：

```text
从 -35 deg 到 +35 deg 全过程都能保持 0.03 N
```

### 5.3 局部排液边界尚未建立

当前缺少：

```text
局部入口
局部出口
左侧排液边
右侧排液边
```

因此不能真实描述泪液在刮擦头附近进入、挤压、侧向排出和后缘空化。

### 5.4 膜厚表达式还需要进一步物理化

当前已修正掉一部分人工膜厚偏置，但最终目标应是：

```text
hfilm = max(h_residual, h0_tear + gap_local)
```

其中 `gap_local` 应来自移动刮擦头表面与角膜前表面的真实局部间隙，而不是用 `lid_mask*delta_h` 制造人工台阶。

### 5.5 摩擦系数尚不能作为最终结果

Stage 200 中液膜摩擦系数为：

```text
muFilm ≈ 0.00639
```

但这只是液膜剪切贡献。最终摩擦系数应包括：

```text
mu_total = (Ffilm_shear + Fboundary_shear) / 0.03 N
```

在局部膜区和排液边界修正前，动态摩擦系数不应作为最终文献对比结果。

## 6. 与最终目标的差距

最终目标是建立一个局部曲面混合润滑动态刮擦模型：

```text
刮擦头从 -35 deg 到 +35 deg 运动
泪膜局部流动和排液
JFO 空化
角膜和刮擦头柔性接触
全程 Wfilm + Fn_contact ≈ 0.03 N
输出摩擦系数、应力、接触压力和膜厚变化
```

当前差距如下：

| 目标 | 当前状态 | 差距 |
|---|---|---|
| 局部薄膜轨迹带 | 尚未完成 | 仍需几何分割角膜前表面 |
| 四周排液边界 | 尚未完成 | 需建立入口、出口、左右侧排液边 |
| 全程动态刮擦 | 尚未完成 | 当前仅有静态平衡点 |
| 全程 0.03 N | 静态可实现 | 动态尚未实现 |
| 物理膜厚场 | 部分完成 | 需基于真实局部 gap |
| 摩擦系数 | 可计算液膜部分 | 缺少边界/粗糙峰剪切和可靠动态膜压 |
| 文献对比 | 尚未完成 | 需稳定输出接触压力、应力、摩擦力随角度/时间变化 |

## 7. 后续应做的修正

### 7.1 从 Stage 200 另存新模型

建议新文件名：

```text
386_lid8mm_stage_local_film_track_setup.mph
```

### 7.2 建立局部薄膜曲面轨迹带

将角膜前表面分割为：

```text
局部球面矩形薄膜轨迹带
其余角膜前表面
```

轨迹带建议覆盖：

```text
角度方向：-35 deg 到 +35 deg，两端各加 0.5-1 mm
横向方向：刮擦头宽度 1 mm，两侧各加 0.3-0.5 mm
```

### 7.3 创建命名选择

至少需要：

```text
sel_film_track
sel_film_inlet
sel_film_outlet
sel_film_side_left
sel_film_side_right
sel_cornea_contact_target
```

### 7.4 限制 Thin-Film Flow 作用域

将 Thin-Film Flow 从整个角膜前表面改为：

```text
Selection = sel_film_track
```

### 7.5 设置四周零表压边界

局部薄膜区域四周设置：

```text
Inlet:  pfilm = 0 Pa
Outlet: pfilm = 0 Pa
Side left:  pfilm = 0 Pa
Side right: pfilm = 0 Pa
```

### 7.6 修正膜厚和移动壁面速度

膜厚目标形式：

```text
hfilm = max(h_residual, h0_tear + gap_local)
```

移动壁面速度：

```text
vwall_x = 0
vwall_y = -omega(t)*Z
vwall_z =  omega(t)*Y
```

### 7.7 逐角度验证

先验证：

```text
theta = -34.5 deg
```

通过标准：

```text
Wfilm < 0.03 N
Wfilm + Fn_contact ≈ 0.03 N
四周 pfilm ≈ 0 Pa
压力集中在刮擦头附近
膜厚没有非物理突变
```

再扩展到：

```text
-35, -30, -20, 0, 20, 30, 35 deg
```

最后才恢复完整连续动态刮擦。

## 8. 当前版本的结论

`385_lid8mm_stage200_official_jfo_joint_load_results_Model.mph` 已经完成了静态混合承载模型的核心框架：

```text
柔性接触 + JFO 薄膜 + 可调压入量 + 0.03 N 载荷平衡
```

但它还不是最终动态摩擦模型。最大问题是薄膜流场仍然作用于过大的角膜区域，缺少局部入口、出口和侧向排液边界。

下一步工作的核心不是继续调摩擦系数，而是修正薄膜计算域和边界条件。只有先解决局部排液问题，后续的全程载荷控制、膜厚变化和摩擦系数输出才具有物理意义。
